# CallPage Crowdsale Contract Repository
 
#### Requirements
*  Node >= 9.0.0
*  Truffle >= 4.0.0
*  Solidity >= 0.4.19

#### Install dependencies
```
npm install
```

#### Tests
###### Run all tests
```
npm test
```

###### Run specific test
```
truffle test/testname.js
```
